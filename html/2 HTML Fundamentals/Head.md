```html
<!DOCTYPE html>
<html>
    <head>
        <title>This is title of document</title>
        <link rel="stylesheet" type="text/css" href="mystyle.css">
        <meta http-equiv="refresh" content="3">
        <meta name="author" content="html lessons">
    </head>
    <body>
        hello world
    </body>
</html>
```

>[!info]- head tags
>\<title> determines the tab name
>\<link> adds refrences to external files, like a css file
>\<meta> adds metadata and changes page behavior. this one refreshes the page every 3 seconds.
